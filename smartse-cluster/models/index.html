Models

This module documents the models contained in the models/ directory at the root of this repository. Models are organised by modelling language / format:

Directory Format

models/sysml/

SysML models (e.g. .sysml, .xmi, or tool-specific project files)

models/ssp/

SSP (System Structure and Parameterization) packages

models/modelica/

Modelica models (.mo files)

models/simulink/

Simulink models (.slx / .mdl files)

Each model should be described in a dedicated page covering:

  • Purpose – what the model represents or analyses

  • Context – which use cases / show cases the model supports

  • Dependencies – other models or external data required

  • How to build / simulate – instructions for running or compiling the model

  • Results / Outputs – expected artefacts produced by the model
